# tkrjs-cli

### install 

```
npm i -g tkrjs-cli
```

### use

**create project：**

```
tkr init myApp
```

**create Controller：** 

```
tkr -c <controller name>
```

**create Service：**

```
tkr -s <service name>
```

### start 

```
tkr start <tsconfig-path> <entry-file-path>
```

```
tkr serve <tsconfig-path> <entry-file-path>
```
